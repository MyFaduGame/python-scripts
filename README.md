
# Python Awesome Scripts

Discover an array of Python scripts designed to simplify tasks and enhance productivity. From password generation to PDF text extraction, these scripts cover diverse functionalities. Dive into the world of over 360 curated Python scripts that promise to amaze and streamline your workflow. Explore various automation possibilities, including text processing with Pandoc and audio manipulation. Delve into Python syntax and gain insights into coding best practices with comprehensive code examples. Additionally, access a curated list of Python scripts aimed at automating mundane tasks and simplifying life.


## Installation

Install All the Scripts by cloning it and run the script using python

```bash
  git clone https://www.github.com/myfadugame/python-scripts
  cd python-scripts
  python3 OR python {script_name}.python
```
NOTE Change {script_name} with your script name
